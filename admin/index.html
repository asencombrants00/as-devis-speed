Comment ça marche ?

  1. Tu modifies les catégories, les photos, les objets ou les textes.
  2. Tu cliques sur Publier sur GitHub.
  3. GitHub Pages remet le site à jour automatiquement.
Important : ne mets jamais ta clé GitHub directement dans un fichier du site. Saisis-la seulement sur cet écran de connexion.
